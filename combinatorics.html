Комбинаторика

Комбинаторика - это раздел математики, который изучает различные комбинаторные структуры и способы подсчёта их числа. Этот раздел математики широко применяется в задачах перестановок, сочетаний, размещений, а также в теории вероятности и дискретной математике.

Основные понятия в комбинаторике

  • Перестановка: Упорядоченное размещение элементов. Например, для множества {A, B, C} существует 6 различных перестановок: ABC, ACB, BAC, BCA, CAB и CBA.
  • Сочетание: Неупорядоченный набор элементов. Например, для множества {A, B, C} существует 3 различных сочетания по 2 элемента: AB, AC и BC.
  • Размещение: Упорядоченное выборка элементов без повторений. Например, для множества {A, B, C} существует 6 различных размещений по 2 элемента: AB, AC, BA, BC, CA и CB.
  • Факториал: Факториал числа n, обозначается как n!, равен произведению всех целых чисел от 1 до n.

Примеры кода на C++

Вычисление факториала


#include <iostream>

// Функция для вычисления факториала числа n
unsigned long long factorial(int n) {
    if (n == 0 || n == 1) {
        return 1;
    }

    unsigned long long result = 1;
    for (int i = 2; i <= n; ++i) {
        result *= i;
    }

    return result;
}

int main() {
    int n = 5;
    unsigned long long fact = factorial(n);

    std::cout << n << "! = " << fact << std::endl;

    return 0;
}

Генерация перестановок


#include <iostream>
#include <algorithm>
#include <vector>

int main() {
    std::vector<int> elements = {1, 2, 3};

    do {
        for (int element : elements) {
            std::cout << element << " ";
        }
        std::cout << std::endl;
    } while (std::next_permutation(elements.begin(), elements.end()));

    return 0;
}
