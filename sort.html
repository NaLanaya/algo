Сортировка пузырьком (Bubble Sort)

Сортировка пузырьком является одним из самых простых алгоритмов сортировки. Она работает путем сравнения и обмена соседних элементов до тех пор, пока весь массив не будет упорядочен. В каждой итерации наибольший элемент "всплывает" на своё место.

Пример кода на C++ для сортировки пузырьком:


        void bubbleSort(int arr[], int n) {
            for (int i = 0; i < n - 1; i++) {
                for (int j = 0; j < n - i - 1; j++) {
                    if (arr[j] > arr[j + 1]) {
                        // Обмен элементов, если текущий больше следующего
                        int temp = arr[j];
                        arr[j] = arr[j + 1];
                        arr[j + 1] = temp;
                    }
                }
            }
        }

Сортировка слиянием (Merge Sort)

Сортировка слиянием является более эффективным алгоритмом по сравнению с сортировкой пузырьком. Он использует принцип "разделяй и властвуй". Алгоритм разделяет список пополам, рекурсивно сортирует каждую половину, а затем объединяет их в упорядоченный список.

Пример кода на C++ для сортировки слиянием:


        void mergeSort(std::vector& arr, int left, int right) {
            if (left < right) {
                int mid = left + (right - left) / 2;
                mergeSort(arr, left, mid);
                mergeSort(arr, mid + 1, right);
                merge(arr, left, mid, right);
            }
        }

Почему сейчас сортировками никто не занимается?

Сортировки, такие как пузырьком, редко используются в современном программировании по нескольким причинам: сложность и производительность, наличие более эффективных алгоритмов и готовых библиотек в стандартных языках программирования.
