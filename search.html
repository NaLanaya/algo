Линейный поиск (Linear Search)

Линейный поиск является простейшим алгоритмом поиска. Он проходит по каждому элементу списка поочередно и сравнивает его с целевым значением. Если элемент найден, алгоритм возвращает его позицию.

Пример кода на C++ для линейного поиска:


#include <iostream>
#include <vector>

int linearSearch(const std::vector<int>& arr, int target) {
    for (int i = 0; i < arr.size(); i++) {
        if (arr[i] == target) {
            return i; // Найдено
        }
    }
    return -1; // Не найдено
}

int main() {
    std::vector<int> arr = {2, 4, 6, 8, 10};
    int target = 6;
    int result = linearSearch(arr, target);
    if (result != -1) {
        std::cout << "Элемент " << target << " найден по индексу " << result << std::endl;
    } else {
        std::cout << "Элемент " << target << " не найден" << std::endl;
    }
    return 0;
}

Бинарный поиск (Binary Search)

Бинарный поиск применяется к упорядоченному списку. Алгоритм разделяет список пополам и сравнивает средний элемент с целевым значением. Если значение больше, то поиск продолжается в верхней половине списка, иначе - в нижней. Процесс повторяется до тех пор, пока не будет найден элемент или список будет исчерпан.

Пример кода на C++ для бинарного поиска:


#include <iostream>
#include <vector>

int binarySearch(const std::vector<int>& arr, int target) {
    int left = 0;
    int right = arr.size() - 1;

    while (left <= right) {
        int mid = left + (right - left) / 2;

        if (arr[mid] == target) {
            return mid; // Найдено
        }

        if (arr[mid] < target) {
            left = mid + 1;
        } else {
            right = mid - 1;
        }
    }

    return -1; // Не найдено
}

int main() {
    std::vector<int> arr = {2, 4, 6, 8, 10};
    int target = 6;
    int result = binarySearch(arr, target);
    if (result != -1) {
        std::cout << "Элемент " << target << " найден по индексу " << result << std::endl;
    } else {
        std::cout << "Элемент " << target << " не найден" << std::endl;
    }
    return 0;
}

Бинарный поиск по ответу (Binary Search for Answer)

Бинарный поиск по ответу - это модификация бинарного поиска, применяемая к задачам оптимизации. Вместо поиска конкретного элемента, этот алгоритм находит минимальное (или максимальное) значение, которое соответствует определенным условиям. Поиск продолжается, подстраиваясь под условия и уточняя результат.
