Вероятность

Вероятность - это важный концепт в теории вероятности и статистике, который изучает случайные явления и их вероятные исходы. Вероятность позволяет нам оценивать шансы на различные события и принимать решения на основе этой оценки. Вероятность изучается в различных областях, таких как статистика, теория игр, машинное обучение и другие.

Основные понятия в вероятности

  • Событие: Исход, который мы хотим проанализировать и для которого мы хотим определить вероятность.
  • Эксперимент: Процесс, который может привести к различным исходам, исследуемым как события.
  • Пространство исходов: Множество всех возможных исходов эксперимента.
  • Вероятность события: Вероятность того, что событие произойдет, обычно выражается как число от 0 до 1.
  • Условная вероятность: Вероятность события, при условии, что другое событие уже произошло.

Примеры кода на C++

Симуляция броска монеты


#include <iostream>
#include <cstdlib>
#include <ctime>

int main() {
    // Инициализация генератора случайных чисел
    std::srand(std::time(0));

    // Симуляция броска монеты (0 - орёл, 1 - решка)
    int outcome = std::rand() % 2;

    if (outcome == 0) {
        std::cout << "Выпал орёл." << std::endl;
    } else {
        std::cout << "Выпала решка." << std::endl;
    }

    return 0;
}

Вычисление вероятности события при множестве экспериментов


#include <iostream>

int main() {
    int totalExperiments = 1000; // Общее количество экспериментов
    int favorableOutcomes = 0; // Количество благоприятных исходов
    int event = 4; // Заданное событие (например, выпадение числа 4 на игральной кости)

    for (int i = 0; i < totalExperiments; i++) {
        // Симуляция эксперимента (бросок игральной кости)
        int outcome = std::rand() % 6 + 1;

        if (outcome == event) {
            favorableOutcomes++;
        }
    }

    // Вычисление вероятности события
    double probability = static_cast<double>(favorableOutcomes) / totalExperiments;
    std::cout << "Вероятность выпадения числа " << event << " = " << probability << std::endl;

    return 0;
}
